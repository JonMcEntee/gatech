Instructions for running code:
1. Clone my code repository with git at https://github.com/JonMcEntee/gatech/tree/master/assignment_4
2. If pip is not installed, install pip (https://pip.pypa.io/en/stable/installing/)
3. Run "pip install -r requirements.txt" from your command line. This will install the python packages needed to run the script.
4. Install jupyter notebook (https://jupyter.org/install)
5. Launch Jupyter notebook by typing "jupyter notebook" in the command line
6. Open a web browser and connect to the jupyter notebook server now running on your machine
7. Navigate to taxi.ipynb or frozen_lake.ipynb and open them
8. Run all cells from top to bottom.
9. This will run the code and export the results as csv files
