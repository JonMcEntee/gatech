Instructions for running code:
1. If pip is not installed, install pip (https://pip.pypa.io/en/stable/installing/)
2. Run "pip install -r requirements.txt" from your command line. This will install the python packages needed to run the script.
3. Install jupyter notebook (https://jupyter.org/install)
4. Launch Jupyter notebook by typing "jupyter notebook" in the command line
5. Open a web browser and connect to the jupyter notebook server now running on your machine
6. Navigate to adult.ipynb or blood_donor.ipynb and open them
7. Run all cells from top to bottom.
8. This will run the code and export the results as csv files

Where to find my data:
Adult dataset - http://archive.ics.uci.edu/ml/datasets/Adult
Blood transfusion dataset - http://archive.ics.uci.edu/ml/datasets/Blood+Transfusion+Service+Center
